## Wiedemann-steiner-syndrome-sas-analysis
End-to-end clinical data analysis of Wiedemann-Steiner Syndrome (WDSTS) registry data using SAS, including data standardization, statistical analysis, and visualization of phenotype patterns. 

## Project Overview
This project presents a structured analysis of rare disease registry data for Wiedemann-Steiner Syndrome (WDSTS) using SAS. The study focuses on transforming heterogeneous, multi-source clinical data into a standardized analytical framework to evaluate phenotype distributions and patient characteristics.

The goal of this work is to support improved clinical understanding and recognition of WDSTS, an often underdiagnosed genetic condition, through data-driven insights.

## Background
Rare disease datasets present unique analytical challenges, including small sample sizes, inconsistent variable formats, and incomplete clinical documentation. WDSTS, a genetically heterogeneous condition, requires careful data curation to enable meaningful analysis of phenotypic variability.

This project applies systematic data preprocessing and statistical methods to improve data quality and facilitate reproducible clinical insights.

## Objectives
Standardize and clean heterogeneous clinical variables
Perform descriptive statistical analysis on registry data
Identify patterns in phenotypic presentation
Improve data consistency for downstream clinical research
Support awareness and data-driven understanding of WDSTS

## Methods

Data Processing:
Standardized inconsistent variables (e.g., age at diagnosis in mixed formats such as years/months)
Transformed developmental milestone data into continuous numerical variables
Performed data validation and quality checks

Statistical Analysis:
Descriptive statistics using
PROC FREQ (categorical distributions)
PROC MEANS (summary statistics)

Visualization:
Generated graphical outputs using PROC SGPLOT
Exported visualizations for phenotype and demographic distributions

Dataset Description:

Source: Rare disease registry data

Sample size: ~137 genetically confirmed cases

Data type: Clinical, phenotypic, and demographic variables

 Note: Raw data is not included due to privacy and ethical considerations.

## Results & Insights
Improved data consistency and usability through systematic standardization
Identified key phenotype distribution patterns across patient cohorts
Enabled structured interpretation of clinical variability in WDSTS
Established a reproducible workflow for rare disease data analysis
 
## Reproducibility
All analyses were conducted using SAS scripts included in the code/ directory. The workflow is designed to be reproducible on similarly structured clinical datasets.

## Significance
This project demonstrates the application of data analytics in rare disease research, highlighting the importance of data standardization and statistical analysis in improving clinical understanding and supporting evidence-based decision-making.
